Meu Projeto

Projeto Instagram Clone aplicado pelo curso Hora de Codar foi o meu primeiro projeto Fullstack feito, utilizando Node.js e Express no Back-end fazendo todas as validações do projeto e definindo as rotas. Foi utilizado o MongoDB Atlas para armazenar as informações do projeto e no Front-end utilizamos o React para criar a interface utilizando Redux para realizar o controle dos usuários autenticados.

Tecnologias: React, Node, Express, MongoDB, Redux
